# emacs.d config

> [!note] When starting Emacs for the first time
> Set `LSP_USE_PLISTS` to true to avoid [this issue](https://github.com/emacs-lsp/lsp-mode/issues/3311#issuecomment-1010074041).

```bash
$ LSP_USE_PLISTS=true emacs
```

### Requirements

**npm**

- `typescript`
- `typescript-language-server`
- `copilot-language-server`
